Graphs in this directory describe German numerals.
It is an adaptation of the graphs created by Andreas Neuman and Sebastian Nagel, previously located under Graphs/ directory.

Main changes:
- Added morphological mode in main graph
- Converted subgraphs from UTF-16 LE to UTF-8
- Updated variable $STEM to $STEM$ to match new variable format
